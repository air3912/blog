### summary
自从LSTM诞生以来，产生了许多variants。然后为了进一步理解LSTM，作者从三个方向，对原版加8个变体进行大规模对比分析。结合超参数优化和重要性评估进行了5400次实验，来确定每个组件的重要性，以确定LSTM未来进一步改进和发展方向之类的。
### method

- 超参数优化+fANOVA重要性评估：作者超参数选取用的是Random Search。然后结果使用fANOVA框架评估确定每个超参数的重要性（影响力）。

对于evaluate部分，按以上方法，作者开始实验。

1. dataset
每组数据集分为 training set, validation set, test set.

然后选择了三组数据集，测试三个不同的方向：TIMIT(语音识别)，IAM Online(手写识别)，JSB Chorales(音乐建模)。
2. architectures
TIMIT 和 IAM 使用的是 Bidirectional LSTM (双向 LSTM)，即一层正向、一层反向处理，最后接 Softmax 输出；而音乐任务使用单向 LSTM
3. Variants
没有输入门/遗忘门/输出们/输入激活/输出激活/输入和遗忘门耦合/没有peepholes/全部门递归（？
4. Hyperparameter
在范围内随机搜索，包括Block数量，learning rate, input noise, momentum这些

最后得出结论：1.LSTM是平均表现最好的   2.移除遗忘门和输出激活函数会导致性能大幅下降   3. 将输入门和遗忘门进行耦合，性能没有太大影响却较少了计算开销。   4. 学习率对结果的影响超过2/3，且参数之间几乎相互独立   5. 动量的影响意外的小
### strengths
作者相当于是对前人的工作进行了一个整合，使用超参数优化+fANOVA重要性评估对LSTM的各个部分（包括超参数和架构）进行一个评估打分，得到每个部分的重要性，从中或许可以窥见未来LSTM进一步优化的方向

作者的实验个人感觉最强在进行的次数足够多吧，故而实验结果是很具有说服力的

还有就是对于测试结果，作者供了全局和top 10%两个分析视角，我觉得这个想法很厉害。
